API changes

o code: replace files *.template by embedded cog templates.
  Lines between CCOG[[[cog... and CCOG[[[end]]] are autogenerated and should
  not be modified.  Modify the template CCOG[[[cog...]]] instead.  If generated
  code is modified, a warning will be issued at next call to genmake2.
o code: move python tools to tools/python/.
o new trait parameter grp_aptype, ap_type becomes dependent trait (do not set)
o new parameter nGRplank in GUD_SIZE.h: set to nPlank for grazing loss diag.
o new parameter nPPplank in GUD_SIZE.h: set to nPlank for per-species PP diag.
o replace traits ExportFrac by ExportFracMort and ExportFracMort2 for linear and
  quadratic mortality terms.  ExportFrac still applies to exudation.
  Same changes for trait parameter grp_ExportFrac.
o new CPP option GUD_PART_SCAV_POP.  If undef, particle scavenging is now based
  on POC and scav_R_POPPOC no longer used.
o replace trait diacoc by hasSi and hasPIC; same for trait parameter grp_diacoc.
o move gud_sort_biovol to GUD_TRAIT_PARAMS; rename logvol0base and logvol0inc
  to logvolbase and logvolinc, resp.
